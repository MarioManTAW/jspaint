How to play

  • Use the available tools to try to match the image on the right.
  • Checks are sent with each percentage point of similarity, which can be seen at the bottom right along with the maximum score in logic.
  • The Archipelago Text Client is accessible under the Extras menu.

Additional features

  • Use File->Open or Ctrl+O to use a custom image. The image will be stretched to fill the canvas and if transparency is present, will overlay the image set in the YAML.
  • Use Extras->Difference Mode or Ctrl+M to visualize the differences between your image and the target.
  • The Pick Color tool, once unlocked, can be used on the target image as well as on the main canvas.
